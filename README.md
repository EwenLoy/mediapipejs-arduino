# 🤖 RoboHand - Управление роботизированной рукой через MediaPipe

![Arduino](https://img.shields.io/badge/Arduino-Uno-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-HTML5-yellow)
![MediaPipe](https://img.shields.io/badge/MediaPipe-Hands-green)
![License](https://img.shields.io/badge/License-MIT-orange)

Проект для управления роботизированной рукой с помощью компьютерного зрения. Отслеживает движения пальцев через камеру и передает команды сервоприводам Arduino в реальном времени.

## 🎯 Демонстрация

```
Камера → MediaPipe → Вычисление углов → Arduino → Сервоприводы → Рука
```

## ✨ Возможности

- 📹 **Отслеживание руки в реальном времени** через MediaPipe
- 🤚 **Поддержка левой и правой руки** с автоматическим определением
- 📡 **Передача данных на Arduino** через Web Serial API
- 🎛️ **5 независимых сервоприводов** для каждого пальца
- 📊 **Визуализация скелета руки** с углами сгибания
- 🔄 **Плавная фильтрация** движений для стабильной работы
- 📈 **Отображение FPS** и детальная логирование

## 🛠️ Требования

### Оборудование
- Arduino Uno (или совместимая плата)
- 5 микросервоприводов (SG90 или аналоги)
- USB кабель для подключения Arduino
- Веб-камера (встроенная или USB)

### Программное обеспечение
- **Chrome/Edge 89+** (для Web Serial API)
- **Arduino IDE** для загрузки кода
- Локальный веб-сервер (для запуска приложения)

## 🚀 Быстрый старт

### 1. Настройка Arduino

1. Откройте `arduino_robohand_simple.ino` в Arduino IDE
2. Подключите сервоприводы к пинам:
   ```
   Большой палец    → Pin 3
   Указательный     → Pin 5  
   Средний          → Pin 6
   Безымянный       → Pin 9
   Мизинец          → Pin 10
   ```
3. Загрузите код на Arduino Uno

### 2. Запуск веб-приложения

1. Скачайте и распакуйте проект
2. Запустите локальный сервер:
   ```bash
   # Используя Python
   python -m http.server 8000
   
   # Или используя Node.js
   npx serve .
   ```
3. Откройте в браузере: `http://localhost:8000`

### 3. Подключение и работа

1. Нажмите **"Подключить Arduino"** и выберите COM-порт
2. Включите камеру кнопкой **"🎥 Включить камеру"**
3. Выберите руку для отслеживания (левая/правая)
4. Нажмите **"▶ Включить передачу"** для отправки данных на Arduino
5. Показывайте руку камере - роботизированная рука будет повторять движения!

## ⚙️ Настройки

### Скорость передачи данных

В настройках интерфейса можно выбрать Baud Rate:
- **9600** - медленно, но надежно
- **115200** - оптимально (по умолчанию)
- **230400** - максимально быстро

### Параметры отслеживания

В `js/app.js` можно настроить:
```javascript
const smoothingFactor = 0.15;    // Сглаживание движений (0.0-1.0)
const minAngleChange = 3;        // Минимальное изменение для отправки
```

## 📡 Протокол связи

Arduino принимает команды в формате:
```
A,B,C,D,E\n
```

Где:
- **A** - угол большого пальца (0-180°)
- **B** - угол указательного пальца (0-180°)
- **C** - угол среднего пальца (0-180°)
- **D** - угол безымянного пальца (0-180°)
- **E** - угол мизинца (0-180°)

Пример команды: `45,120,90,60,30\n`

## 🔧 Устранение проблем

### Камера не работает

1. **Проверьте браузер:** Используйте Chrome/Edge 89+
2. **Разрешите доступ:** Браузер запросит разрешение на камеру
3. **Проверьте другие приложения:** Закройте Skype, Zoom, Teams
4. **Проверьте камеру:** Откройте https://webcamtests.com/

### Arduino не подключается

1. **Проверьте порт:** Убедитесь, что Arduino подключена к USB
2. **Закройте Serial Monitor:** В Arduino IDE закройте монитор порта
3. **Проверьте драйверы:** Установите драйверы CH340/CP2102 если нужно
4. **Перезагрузите Arduino:** Нажмите кнопку Reset

### Движения дёргаются

1. **Увеличьте сглаживание:** `smoothingFactor = 0.3`
2. **Увеличьте порог:** `minAngleChange = 5`
3. **Проверьте освещение:** Хороший свет улучшает отслеживание

## 🛠️ Модификации

### Добавление новых режимов

В `js/app.js` можно добавить:
```javascript
// Режим "зажать/отпустить"
if (isGrabMode) {
    angles = angles.map(a => a > 90 ? 180 : 0);
}
```

### Калибровка сервоприводов

В Arduino коде:
```cpp
// Калибровочные значения для каждого сервопривода
const int calibrations[5] = {0, 0, 0, 0, 0};
servos[i].write(angles[i] + calibrations[i]);
```
